# Discover Your Direction

Discover Your Direction 是一个中文优先的 Codex Skill：它从你的真实经历，以及你可选、明确指定的本地证据中，提出三个个人方向假设，帮助你选择一项可证伪的身份实验，再用七天的现实反馈减少不确定性。

## 它不是什么

- 不是人格测评、职业测评或心理诊断。
- 不保证在十五分钟内找到终身使命、完美定位或唯一答案。
- AI 推断不是事实；你可以随时纠正、否定或补充它的解释。

## 能做什么

- 通过一次只问一个问题的证据访谈，回到具体事件、行动、结果、反馈与反例。
- 可选地对你明确指定路径的本地文件做只读分析；不启用文件分析也能完成访谈。
- 区分“来源观察”和“当前 AI 推断”，并为来源观察保留文件定位，或 `current conversation` 中的回合/消息定位。
- 生成三个实质不同的可能方向，分别列出最大未知和针对具体假设组件的可观察推翻条件，而不是用一次结果否定整个身份。
- 为内容、服务、咨询、工具、工作坊、研究、手艺等不同工作方式设计与真实角色相符的七天实验，不默认把所有方向变成内容创作。
- 支持从已有方向或实验继续，也支持带着执行结果返回复盘；没有预注册的结果会被明确标为事后解释。

## 安装

使用前提：你需要一个支持 Skills 的 Codex 环境。安装位置会随 Codex 版本和安装器而异；可先查看当前的 [Codex Skills 官方文档](https://developers.openai.com/codex/skills)。

手动安装：

1. 按当前官方文档，优先把仓库子文件夹 `discover-your-direction/` 复制到 `$HOME/.agents/skills/discover-your-direction`；Windows 写法是 `$HOME\.agents\skills\discover-your-direction`。
2. 兼容性提示：部分 Codex 版本或内置安装器仍使用 `$CODEX_HOME/skills/discover-your-direction`，未设置 `CODEX_HOME` 时默认为 `~/.codex/skills/discover-your-direction`。请以你安装的 Codex 或安装器实际报告的目标位置为准。
3. 新开一个 Codex 会话；如果 Skill 没有出现，重启 Codex 后再试。
4. 使用下方的完整示例调用，并确认 Skill 会让你选择直接访谈或分析你明确指定的文件。

从 GitHub 安装时，可以请 Codex 从仓库子文件夹 URL 安装这个 Skill：<https://github.com/taotaotao07/discover-your-direction/tree/main/discover-your-direction>。这里不提供未经验证的命令行语法；请按你的 Codex 环境所支持的安装流程操作。

## 使用

```text
$discover-your-direction 帮我从过去的经历里找到值得验证的下一步方向。
```

你可以直接开始访谈，也可以指定希望分析的本地文件。选择文件方式时，Skill 会先解析路径，展示即将读取的完整、确切文件清单，说明访问仅为只读，并在得到你的明确肯定后才读取；范围变化时会重新征得同意。

## 隐私

本地资料分析完全可选。Agent 只把你确认的文件用于读取和分析，不会修改源文件，也不会自行跟随清单之外的链接、附件或引用。你始终亲自决定并执行发送、发布、招募、联系和交付等外部行动；Agent 只起草你可以选择使用的材料。

这里的“本地文件”只描述资料的来源位置，并不表示内容一定只在本机处理。资料如何被处理、传输和保留，取决于你使用的 Codex 产品、模型提供方、账户及组织设置。你必须检查适用于当前环境的政策与设置；使用 OpenAI 产品时，可参阅 [OpenAI 隐私政策](https://openai.com/policies/privacy-policy/) 和 [数据如何用于改进模型](https://help.openai.com/en/articles/5722486-how-your-data-is-used-to-improve-model-performance)。

请勿提供密钥、密码、账户数据，或任何你不希望被处理的私人材料。若范围过大、内容可疑或包含敏感信息，应缩小范围或跳过文件分析。

## 方法

这里的“方向”是等待现实检验的假设，不是身份裁决。约十五分钟的核心对话用于整理证据、提出三个可能方向并选择一个实验；这不是强制倒计时，较多资料的读取时间也不计入其中。接下来的七天用于收集与角色相符的外部信号、真实执行后的内部体验、技能成长，以及矛盾和意外。

实验没有通用的承诺阶梯：付费、复用、同行审阅、采用、持续练习或认真参与，只有放进具体角色和场景里才有意义。所有外联、发布、招募、联系和实际交付都由用户本人执行。

## 局限与安全边界

这个 Skill 只能帮助形成和检验假设，不能替代专业判断，也不保证职业或商业结果。医疗、法律、金融、治疗、未成年人保护等受监管或涉及脆弱人群的场景，不应通过无资质建议或服务来实验；应改用公开资料研究、合格从业者访谈或合格专业人士的监督。相关观察必须是公开观察，或者同时获得适当同意与必要的机构或专业批准。
